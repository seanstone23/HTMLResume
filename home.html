This first project was when I was tasked with creating a 3D mobile game using Unity. I chose to create a 3D horror game, heavily influenced by the popular slender man game from 2011. This was my first time having free range in making my own game so I learnt a large amount about C# scripting during this project. The game was originally created for a mobile platform however I ported it to PC so that I could create a video. You can still see the remnants of the mobile controls on screen. This project scored me a first!





In this project, I had been tasked to create a game of my choosing. I chose to use the Unity engine to create a 2D real time strategy game. My reason for choosing this style is that I hadn't had much experience using Unity 2D a part from a small group project within the first year. I wanted to expand my skillset and create something I would enjoy playing. I refreshed my skills in C# scripting and learnt different techniques including animation and tilemapping.





This next video demonstrates another project I had been tasked with during my studies. This was to utilise C++ programming and the BOX 2D library in order to create a physics based sport game. I chose to do crazy golf as I believed that I would be able to demonstrate the physics with all of the different obstacles the player would have to overcome. This was quite a difficult task and unfortunately I may have been too ambitious as I ran out of time to complete the second level. However, due to the fact I utlised physics well and enabled the user to use multiple clubs to differentiate the shots, I recieved a good mark for this.





This is a video demonstrating my work during an animation module at University. I was tasked within creating a short animation using Maya. Using a pre-rigged model, I managed to animate this figure to move to different poses. I learnt a large amount about actually rigging models, and how to use Key Frames to animate. I also learnt a large amount about how to actually use Maya.
